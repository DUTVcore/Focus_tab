## ğŸš€ Installation

The project is currently in **Developer Preview**. You can install it manually ("Side-load") as follows:

1.  **Clone this repository to your machine:**
    ```bash
    git clone https://github.com/DUTVcore/Focus_tab.git
    ```
2.  Open Chrome (or Edge/Brave), type into the address bar:
    `chrome://extensions/`
3.  Enable **Developer mode** (Top right corner).
4.  Click **Load unpacked** -> Select the `Focus_tab` folder you just cloned.
5.  Open a New Tab and enjoy!
